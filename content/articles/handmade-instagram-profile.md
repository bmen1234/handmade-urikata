## ハンドメイド作家のInstagramプロフィール最適化

Instagramはハンドメイド作家にとって最重要の集客媒体です。**フォロワー数と月商には明確な相関があり（500人=月5万、1,000人=月20万、2,000人以上=月266万）**、プロフィールの設計次第でフォロワーの増加速度が大きく変わります。この記事では、ハンドメイド作家のためのInstagramプロフィール最適化を解説します。

---

## なぜInstagramプロフィールがそれほど重要なのか

Instagramでは、投稿を見た人がプロフィールページに訪問し、フォローするかどうかを数秒で判断します。

プロフィールを整えていない作家と整えている作家では、**同じ投稿クオリティでもフォロー転換率に3〜5倍の差が生まれる**ことがあります。

プロフィールは「あなたのショップの看板」です。訪問者が「この人は何を作っている人か」「フォローしたら何が見られるか」を一瞬で判断できるように設計しましょう。

---

## Instagramプロフィールの5つの最適化ポイント

### 1. ユーザーネーム（@アカウント名）の設計

ユーザーネームはminneのショップ名と統一することを強くお勧めします。**SNSとECを行き来するユーザーが「あ、さっきInstagramで見た人だ」と気づけるようにする**ためです。

- 短くて読みやすい（記号は最小限）
- ショップ名・ブランド名と同じか近い
- 変更を頻繁にしない（URL共有した時に壊れる）

### 2. プロフィール写真の選び方

- **作品の写真**：代表作・最も映える1点を使う（ブランドイメージを伝えやすい）
- **顔写真**：親近感・信頼感を高める（作家の人柄を見せたい場合）

どちらでも構いませんが、**「作品を全面に出すアカウント」か「作家の人柄を見せるアカウント」かを統一する**ことが重要です。途中で変えると世界観が崩れます。

### 3. 名前（表示名）に検索キーワードを入れる

Instagramの「名前」フィールドは検索対象になります。**ジャンルキーワードを入れることで検索流入が増えます。**

例：
- 「〇〇 / レジンアクセサリー作家」
- 「△△ハンドメイド / 刺繍・布小物」
- 「×× / ドライフラワースワッグ」

名前にジャンルを明記することで、「レジン作家をフォローしたい」という能動的なユーザーからフォローされやすくなります。

### 4. 自己紹介文（bio）の書き方

140文字以内のbioで伝えるべき情報は以下の通りです。

**BIOの黄金テンプレート：**
```
①何を作っているか（ジャンル・素材）
②誰のために作っているか（ターゲット）
③どんな想い・こだわりで作っているか
④購入先URL（minne・BASE等）
```

例：
```
押し花レジンアクセサリー作家🌸
季節の花を閉じ込めた一点物ピアス
化学素材を使わない天然素材にこだわり
▼作品はminneで販売中
```

### 5. プロフィールリンクの最適化

Instagramのbioリンクは1つしか設定できません。**Linktree・lit.linkなどのリンクまとめツールを使って複数の誘導先を設定しましょう。**

- minne（メイン販売先）
- Creema（サブ販売先）
- ワークショップ申し込みページ
- ブログ・note

## ハイライト（ストーリーズ保存）の設計

プロフィールページには「ハイライト」と呼ばれるストーリーズの保存機能があります。初めて訪問した人向けの情報を整理して保存しましょう。

おすすめのハイライト構成：

| ハイライト名 | 内容 |
|------------|------|
| 作品紹介 | 代表作品の画像・動画 |
| 制作工程 | 作業動画・ビフォーアフター |
| お客様の声 | 購入者からのレビュー・写真 |
| 購入方法 | minneへの誘導・支払い方法 |
| 作家について | 自己紹介・制作へのこだわり |

ハイライトは**新規フォロワーが「この作家はどんな人か」を調べるときに最初に見る場所**です。丁寧に設計しましょう。

## フィード投稿の世界観を統一する

プロフィールページを見たとき、投稿の色合い・雰囲気が統一されていると**「プロフェッショナル感」が生まれ、フォロー率が上がります。**

統一の方法：
- **フィルターを固定する**：Lightroomのプリセットを1つ決めて全投稿に適用
- **背景色を統一する**：白・ベージュ・グレーなど1〜2色に絞る
- **撮影場所を統一する**：同じ場所・同じ光源で撮影する

## まとめ

ハンドメイド作家のInstagramプロフィール最適化のポイントをまとめます。

1. **ユーザーネームはショップ名と統一する**
2. プロフィール写真は「作品派」か「人物派」で統一する
3. **名前フィールドにジャンルキーワードを入れる**
4. BIOに「何を・誰のために・こだわり・購入先」を盛り込む
5. Linktreeで複数の販売先へ誘導する
6. ハイライトで新規訪問者向けの情報を整理する
7. フィード投稿の世界観・色調を統一する

**プロフィールはSNS集客の入口です。** 週1回投稿しても、プロフィールが整っていなければフォローされません。まずプロフィールを整えることから始めましょう。
